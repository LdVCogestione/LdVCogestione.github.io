cogestione

Cogestione è un software per l'organizzazione delle cogestioni scolastiche ideato e sviluppato da Pietro De Nicolao. È stato usato per diversi anni dal Liceo Scientifico Albert Einstein di Milano. Il software consente agli studenti di prenotarsi a delle attività che si svolgono in parallelo.

Le prenotazioni

Gli utenti si prenoteranno con un'interfaccia web.

Altamente configurabile

L'applicativo è configurabile con una comoda interfaccia. È possibile specificare le attività, le classi, gli orari di apertura e chiusura delle prenotazioni e anche un blacklist di nomi.

Download, installazione e utilizzo

I link per il download sono disponibili sulla colonna destra della pagina. Per informazioni sull'installazione e l'utilizzo del software, leggere il file README.

Features

  • Attività riservate a quarte e quinte
  • Blacklist
  • Gestione dell'affollamento: numero massimo di posti per ogni attività
  • Liste di prenotati
  • Attivazione e disattivazione automatica delle prenotazioni
  • Possibilità di cancellare le prenotazioni via web
  • Attività configurabili via web
