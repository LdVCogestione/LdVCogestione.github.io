cogestione

Questo è un software per l'organizzazione delle cogestioni scolastiche. In particolare, consente agli studenti di prenotarsi a delle attività che si svolgeranno contemporaneamente.

Le prenotazioni

Gli utenti si prenoteranno con un'interfaccia web.

Altamente configurabile

L'applicativo è configurabile con una comoda interfaccia. È possibile specificare le attività, le classi, gli orari di apertura e chiusura delle prenotazioni e anche un blacklist di nomi.

Features

  • Attività riservate a quarte e quinte
  • Blacklist
  • Classi configurabili
  • Gestione dell'affollamento
  • Liste di prenotati
  • ...
